
---( This is not a mod )---

Blaze models for MineClone2 that rotate and dance more like MineCraft's
by Bioman

The -v1 is a version that uses four bones for the turning dance animation
The -v2 uses additional bones inside the rotating sticks for the front facing effect

Either model can be used the choice is yours

To use, replace all 4 files in
MineClone2\mods\ENTITIES\mobs_mc

Skin edited from Adventurecraft texture pack.
